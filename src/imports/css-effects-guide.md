🎬 1. ANIMAÇÕES DE SCROLL (Scroll Reveal)
Elementos aparecem quando entram na tela:

Fade In - Aparecer suavemente (opacidade 0 → 1)
Slide In Bottom - Deslizar de baixo pra cima
Slide In Left - Deslizar da esquerda
Slide In Right - Deslizar da direita
Slide In Top - Deslizar de cima pra baixo
Scale In - Crescer do pequeno pro normal
Rotate In - Girar enquanto aparece
Bounce In - Aparecer com "pulo"
Flip In - Virar como uma carta
🌑 2. SOMBRAS
Box Shadow - Sombra ao redor do elemento
Text Shadow - Sombra no texto
Drop Shadow - Sombra que segue formato do elemento
Inner Shadow - Sombra interna
Glow Effect - Brilho (sombra colorida, tipo neon)
🖱️ 3. EFEITOS DE HOVER (ao passar o mouse)
Scale Up/Down - Crescer/diminuir
Brightness - Ficar mais claro/escuro
Rotate - Girar
Translate - Mover (cima/baixo/lados)
Opacity Change - Mudar transparência
Color Change - Mudar cor
Border Grow - Borda aparece/cresce
Underline Expand - Sublinhado expande
Lift (elevate) - Elemento "levanta" (shadow + translate)
🎨 4. FILTROS CSS
Blur - Desfocar
Grayscale - Preto e branco
Brightness - Brilho
Contrast - Contraste
Saturate - Saturação de cores
Hue Rotate - Girar cores
Invert - Inverter cores
Sepia - Tom sépia (vintage)
🔄 5. TRANSFORMAÇÕES
Scale - Tamanho (maior/menor)
Rotate - Rotação (graus)
Translate - Posição (mover X/Y)
Skew - Inclinação/distorção
3D Rotate - Rotação 3D (X/Y/Z)
Perspective - Perspectiva 3D
✨ 6. EFEITOS MODERNOS
Glassmorphism - Efeito vidro fosco (backdrop-blur)
Gradient Animation - Gradiente animado
Shimmer/Shine - Brilho que passa
Particle Effects - Partículas
Parallax - Movimento em camadas (scroll)
Magnetic Hover - Elemento "puxa" cursor
💫 7. TRANSIÇÕES (suavização de mudanças)
Velocidade - Rápida/Normal/Lenta
Easing - Linear, Ease, Ease-in-out, etc.
Delay - Atraso pra começar
👁️ 8. OPACIDADE
Normal - Transparência padrão
Hover - Transparência ao passar mouse
